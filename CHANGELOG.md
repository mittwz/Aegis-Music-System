# Changelog

## 4.0.0

- renamed the project to **Aegis Music System**
- switched the default project language to English
- added `README.pt-BR.md` as an optional Portuguese alternative
- added a root `dota_gsi/` folder with the ready-to-copy Dota GSI file
- removed separate output presets and replaced them with a single `master_gain_percent`
- kept and refined **cinema mode**
- made `config.json` comment-friendly through JSON-with-comments loading
- improved state labels and documentation for GitHub release use
- kept both creator-safe and copyrighted playlist profiles
